# Satellite Image Denoising

IEEE Published research on satellite image denoising using 
machine learning. Outperforms Gaussian, Median, and Wavelet 
filtering techniques.

## Live Demo
Try it instantly — no setup needed:
👉 https://huggingface.co/spaces/Balamaneesh2520/satellite

## Results
- PSNR: 38.17 dB
- SSIM: 0.9648
- MSE: 9.90

## Tech Stack
Python · OpenCV · NumPy · scikit-image · Gradio · HuggingFace
